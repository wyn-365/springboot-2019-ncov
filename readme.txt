1.创作初心：
2020年春节前后，新型冠状病毒肺炎在武汉爆发并迅速蔓延全国，日本、韩国、新加坡和泰国等国家也陆续出现了一定规模的感染病例。本次疫情爆发突然，病毒感染性高，涉及范围广，形势严峻。
疫情当前，共克时艰。新型冠状病毒肺炎疫情牵动着全国人民的心，面对来势汹汹的疫情，各地政府积极响应应对，但由于应急疫情管理本身的分散性，疫情早期异常数据未能及时上报公示，造成疫情防控工作开展延误，各渠道数据不能及时整合共享，各个单位难以统筹协同，影响疫情防控效率。
作为计算机技术和软件工程专业研究生，我们希望能够尽所学积极参与到抗击新型冠状病毒肺炎疫情中，应用专业技能及大数据技术优势，搭建疫情数据可视化平台，多渠道、多角度向公众展现最新疫情态势，进一步完善疫情防控体系、提高疫情防控效率，为奋战在一线的防疫人员提供最直观的疫情分析，助力疫情防治工作，提供实时、可靠、可理解、易沟通的疫情信息，为打赢此次防疫攻坚战贡献自己的力量。
我们的初衷在于让疫情大数据可视化走进人群，现在手机用户量基数大，如同每个人的身份证一样，所以对于兼容移动端时非常重要的一件事，我们的作品不仅可以使科技人员分析疫情趋势和直观的结果，更可以让各阶层，各个年龄段的人看懂疫情，了解疫情，防控疫情。
应用场景：
1）疫情数据实时监测。基于丁香医生、腾讯和天行API实时同步的疫情数据，实时追踪疫情进展情况；
2）疫情信息协同共享。整合多渠道最新疫情数据、资源调度等各类防疫信息，及时公开共享，有效实现跨平台多源数据共享；
3）防控知识科普。客观完整还原疫情发展现状，提供官方验证的病理知识、防疫方法、辟谣防护信息，为科学防御提供理论支持，防止谣言滋生，稳定全社会抗击疫情信息。

3.作品说明（如：作品介绍、社会价值、面向对象等）
作品介绍：小疫防控通过数据可视化平台应用于我国突发疫情防控领域，不仅面向数据分析人员，而且面向基数最大的手机用户群，基于丁香医生、腾讯和天行API实时新型冠状病毒肺炎数据，分析最新疫情发展态势，整合多渠道疫情信息，采用大数据信息展图直观展现全国疫情分布、多维信息和疫情走势，为公众和一线防疫人员提供直观的呈现、理解、分析、预测和防控工具。疫情数据可视化平台应用于新型冠状病毒肺炎的疫情数据实时监测、疫情信息协同共享和防控知识科普等场景，展现全国疫情实时数据动态、实现疫情信息实时播放。
公众可通过疫情数据可视化平台查询全国及各省市疫情实时数据、了解疫情发展态势、获悉周边疫情、学习科学防疫知识。平台也可协助一线防疫人员，提供大数据可视化列表，动态展现疫情期间湖北省人员流动情况、湖北省物资供应情况、全国各省市确诊治愈散点分布、各省市地区地理信息确诊治愈情况及全国确诊、疑似病例趋势图。
社会价值：近日来，面对新型冠状病毒肺炎，各行各业都在行动。疫情数据可视化平台以大数据技术为依托，进一步优化集成了新冠肺炎疫情防控动态信息，跟进新冠肺炎最新科普动态，实时向公众及一线防疫人员展示最新疫情态势，进一步完善疫情防控体系、提高疫情防控效率，为打赢疫情防控阻击战提供强有力的科技信息支撑服务。
面向对象：社会公众及一线防疫人员。
4.作品价值及影响力
疫情数据可视化平台作为基于真实疫情数据，整合分析全国疫情发展态势，通过大数据展图，向公众及一线防疫人员快速直观的呈现全国疫情分布、物资供应情况、全国各省市确诊治愈分布及全国确证/疑似病例发展趋势。
将疫情数据可视化平台应用于新型冠状病毒肺炎的疫情数据实时监测、疫情信息协同共享和防控知识科普等场景，可实时追踪疫情数据更新情况，增加疫情信息的多渠道整合共享，实现防疫管理工作的统筹协同，提高公众科学防疫意识，用科技助力新型冠状病毒肺炎防控工作，为疫情防控工作的信息公开、科学防疫知识普及提供真实可信的科学依据，在提高疫情防控效率的同时，增加政府公信力。此次新型冠状病毒肺炎疫情过后，疫情数据防疫平台亦可应用在更多疫情应急防控工作相关的场景中，为健全我国国家疫情防控体系、完善突发疫情防控管理工作、提升疫情管理能力做出更多贡献。
5.作品技术说明
数据来源：定时更新同步于丁香数据、腾讯数据、天行数据API；
数据集处理：对网络对接的数据集采用及时入库的方法，在程序中加以定时的处理方式对丁香数据和天行数据API返回结果加以核对、入库以及实时更新，并对历史数据集进行保持久化；
数据处理技术：本作品采用多种数据源，多种数据处理技术，如json，xml,序列化对象等；
数据处理方法：对于部分可视化中的页面，进行ajax异步交互，能够做到不刷新页面的情况下进行更新数据；
数据处理工具：
1）阿里巴巴fastjson处理工具解析json数据；
2）dom4j文档操作；
3）java原生的序列化工具将对象在网络上进行传输；
4）httpclient发送http网络请求及和jsoup处理html数据；
5）mybatis对象与数据库表的映射orm；
6）echarts可视化方法；
7）模板引擎themeleaf。
前台：echarts、js、amazeui、bootstrap、html；
后台：mybatis；
框架：spring、springmvc、springboot；
开发语言：java
数据库：mysql；
数据源：Druid连接池；
服务器：阿里云服务器linux（centos7.6）。
